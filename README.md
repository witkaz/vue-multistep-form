# multistep-form

Wizard Multistep Form built with Vue, Vuelidate, v-select and axios.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
